Использование плагинов

Установка

Для установки плагина из .zip / .rar архива проделайте следующее:

  • В UV откройте диалог "Опции -- Настройки плагинов", нажмите "Добавить...", выберите опцию "Установить плагин(ы) из архива(ов)" и нажмите кнопку "...".
  • В открывшемся диалоге выберите файл архива (или несколько файлов).
Либо, если архив содержит файл "pluginst.inf":
  • Просто откройте архив в UV. Появится диалог авто-установки.
Примечание: Учтите, что установка новой версии плагина, из архива с другим именем, может не перезаписать старую установку. Поэтому старую версию плагина придется удалить из списка вручную.

Настройка

Вы можете настроить использование плагина через т.н. "детект-строку". В диалоге "Настройки плагинов" нажмите "Настроить...", там можно отредактировать детект-строку.

Детект-строка определяет, какие файлы будут открываться в плагине по умолчанию. Если эта строка пуста, то плагин должен будет определять, какие файлы ему открывать, по содержимому файла. Строка может содержать следующее:

  • Список расширений файлов в виде:
    EXT="TXT" | EXT="DOC" | EXT="RTF"
  • Специальное слово "FORCE", вместе с расширениями или без:
    EXT="TXT" | FORCE
    Это слово означает, что плагин также будет открываться для всех неопознанных файлов при ручном вызове режима Плагины (клавиша "7").
  • Список расширений файлов в виде:
    EXT:TXT,DOC,RTF
    EXT:TXT FORCE
    Слово "FORCE" может присутствовать, после пробела, в конце строки.

Использование

  • Можно использовать клавишу "7" для переключения между всеми плагинами, подходящими для данного файла.
  • Можно видеть, какой плагин активен в данный момент, в заголовке окна программы: имя плагина там написано в скобках.
