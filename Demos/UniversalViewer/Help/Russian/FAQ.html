В1: Как интегрировать Universal Viewer в популярные файловые менеджеры?

О:

  • Проводник Windows

    В UV откройте диалог "Опции -- Настройки", 1-я вкладка, отметьте опцию "Пункт в контекстном меню Проводника". (В Vista/Win7 может потребоваться запустить UV под учетной записью Администратора.) Это добавит пункт UV в контекстное меню Проводника.

  • Total Commander

    Диалог "Конфигурация -- Настройка", закладка "Правка/Просмотр", отметить "Внешнюю программу просмотра" и ввести путь к файлу Viewer.exe в поле "по умолчанию:". Если путь содержит пробелы, нужно заключить его в двойные кавычки.

    Это настроит вызов UV на клавишу F3.

    Также можно создать кнопку на панели инструментов TC, чтобы открывать список файлов, выделенных на файловой панели. Перетащите файл Viewer.exe на пустое место панели инструментов, создастся новая кнопка. Щелкните на ней правой кнопкой мыши, выберите Изменить. В диалоге параметров в поле Параметры введите "@%UL" (с кавычками).

  • FAR Manager

    Диалог "Параметры -- Настройки программы просмотра", отметить "Запускать по Alt+F3" и ввести путь к файлу Viewer.exe с дополнительным макросом в поле "Команда просмотра":
    "C:\Program Files\Universal Viewer\Viewer.exe" "!\!.!"

    Это настроит вызов UV на клавиши Alt+F3.

    Также можно передавать UV список файлов, отмеченных на панели FAR. Чтобы добиться этого, немного измените значение поля "Команда просмотра":
    "C:\Program Files\Universal Viewer\Viewer.exe" "@!@AF!"

  • FreeCommander

    Диалог "Разное -- Параметры", закладка "Программы", ввести путь к Viewer.exe в поле "Внешний просмотрщик".

    Это настроит вызов UV на клавиши Shift+F3.

    Чтобы настроить вызов UV на F3, зайдите в диалог "Разное -- Менеджер горячих клавиш", и для команды "Файл - Просмотр внешний" установите горячую клавишу F3.

  • SpeedCommander

    Диалог "Extra -- Settings", закладка "Advanced -- Applications & Internet", ввести путь к Viewer.exe в поле "Viewer:".

    Это настроит вызов UV на клавишу F3.

В2: Не просматриваются документы DOC/XLS, ODT/ODS, PDF. Что требуется установить для этого?

О1:

Для показа DOC/XLS следует установить одно из перечисленного:

  • Microsoft Office
  • Lister-плагин для DOC/XLS
Для показа ODT/ODS/ODP следует установить одно из перечисленного:
  • OpenOffice.org
  • Lister-плагин для ODT/ODS
Для показа PDF следует установить одно из перечисленного:
  • Adobe Reader
  • Foxit Reader
  • Lister-плагин для PDF

О2: Если установка офисных просмотрщиков (из ответа выше) не помогает, и окно UV в режиме Интернет остается пустым, попробуйте включить опцию "Мультимедиа -- Интернет -- Открывать неизвестные типы файлов".

В3: Каковы ограничения на размер просматриваемых файлов?

О: Программа может показывать, в режимах Текст/Двоичный/Шест./Юникод, файлы неограниченного размера. Вы можете быстро прокручивать даже огромные файлы, поскольку программа подгружает в память только необходимую часть файла (где-то 120-180 Кбайт), а не весь файл целиком.

Однако, это неверно для режима RTF/UTF-8 и (в большинстве случаев) для режима Плагины. RTF-режим и большинство плагинов грузят в память все содержимое файла, что для файлов, слишком больших по размеру, очень медленно или вообще невозможно.

В4: Насколько безопасно открывать ссылки на HTML страницах в режиме Интернет?

О: Безопасность этого зависит от текущей конфигурации MSIE и набора "заплаток" к нему, установленных в системе.

Если Вы боитесь открывать какую-то ссылку в самом MSIE, Вы так же не должны открывать ее в UV. Рекомендуется включить опцию "Вид -- Интернет -- Автономный просмотр", что запретит просмотр online-контента в окне UV. Только учтите, что эта опция не влияет на активацию ссылок, у которых задано свойство "Открывать в отдельном окне".

В5: Можно ли отображать Flash-анимацию не развернутой во все окно? Хотелось бы показывать ролики в размере 100%, 200% или другом.

О: По умолчанию Flash-анимация показывается в режиме Интернет, в окне MSIE. Можно показывать ее другим способом, при котором можно регулировать размер ролика:

  • в окне Windows Media Player. Для этого зайдите в диалог настроек и на первой закладке переместите расширение "swf" из списка режима Интернет в список режима Мультимедиа. После чего также включите режим "Windows Media Player 9.0" на второй закладке диалога.
  • с помощью одного из плагинов.

В6: Почему не показываются некоторые UTF-8 тексты?

О: В старых версиях UV (старее 1.5.9.0) было требование наличия в файле специальной UTF-8 сигнатуры. В новых версиях это не требуется, только учтите, что:

  • UTF-8 не поддерживается в Windows 9x.
  • UTF-8 тексты без сигнатуры не могут быть автоопределены как UTF-8, поэтому на них Вам придется включать режим просмотра "RTF/UTF-8" вручную.

В7: Документы Word/Excel иногда открываются в новом окне...

О: Это известная проблема MS Office, которая может решаться применением следующего .REG-файла. Сохраните приведенный ниже текст в файл с именем "Fix.reg" и запустите этот файл в Проводнике: 

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Word.Document.8]
"BrowserFlags"=dword:80000024

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Word.RTF.8]
"BrowserFlags"=dword:80000024

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Word.Document.12]
"BrowserFlags"=dword:80000024

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Word.DocumentMacroEnabled.12]
"BrowserFlags"=dword:80000024

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Excel.Sheet.8]
"BrowserFlags"=dword:80000A00

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Excel.Sheet.12]
"BrowserFlags"=dword:80000A00

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Excel.SheetMacroEnabled.12]
"BrowserFlags"=dword:80000A00

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Excel.SheetBinaryMacroEnabled.12]
"BrowserFlags"=dword:80000A00

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.Show.8]
"BrowserFlags"=dword:800000A0

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.Show.12]
"BrowserFlags"=dword:800000A0

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.ShowMacroEnabled.12]
"BrowserFlags"=dword:800000A0

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.SlideShow.8]
"BrowserFlags"=dword:800000A0

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.SlideShow.12]
"BrowserFlags"=dword:800000A0

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\PowerPoint.SlideShowMacroEnabled.12]
"BrowserFlags"=dword:800000A0

В8: Можно ли изменить внешний вид элемента управления WMP 6.4?

О: Можно привести вид элемента управления Windows Media Player 6.4 к виду, аналогичному WMP 9.0 (если WMP 9.0+ установлен у Вас в системе). Для этого примените следующий .REG-файл, сохрание текст в файл "WMP.reg" и запустите его: 

REGEDIT4

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MediaPlayer\ShimInclusionList\VIEWER.EXE]

Для отмены этого изменения примените следующий .REG-файл: 

REGEDIT4

[-HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MediaPlayer\ShimInclusionList\VIEWER.EXE]

В9: При получении графики из IrfanView/XnView не работает анимация...

О: Интерфейс IrfanView/XnView - это посылка приложению простого битмепа (bitmap), при этом, конечно же, не поддерживается анимация и пропадает прозрачность. Поэтому, для промотра анимированных GIF переключайтесь в режим использования внутренней графической библиотеки; подсказка: используйте клавишу '4' для переключения между библиотекой и IrfanView/XnView.

В10: Есть ли у программы моменты с Юникодными именами файлов?

О:

  • Не устанавливайте программу в папку с Юникодным именем, это не поддерживается.
  • Юникодные имена могут не поддерживаться для: IJL15.dll, IrfanView, плагинов. В этих случаях файлы с Юникодными именами читаются через короткие имена (8.3). Короткие имена должны быть разрешены в вашей ОС. Чтобы проверить, разрешены ли они в некоей папке, сделайте следующее: запустите "cmd.exe" в этой папке, в нем запустите команду "dir /x" - эта команда должна показать короткие имена для Юникодных имен.

В11: Как просмотреть CHM-файлы?

О: Установите плагин HTMLView. Затем зайдите в диалог настройки плагинов, для плагина HTMLView нажмите кнопку "Настроить", и в поле "Детект-строка" введите:
ext:chm
