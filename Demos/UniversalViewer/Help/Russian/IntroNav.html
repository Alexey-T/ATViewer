Дополнение NavPanel

Дополнение Navigation Panel вызывается из Universal Viewer нажатием F4 (клавиша по умолчанию). Оно позволяет выбирать файлы в интерфейсе Проводника. Файл, который отмечен в NavPanel, показывается в просмотрщике. И файл, который выбирается в просмотрщике, отмечается в NavPanel.

NavPanel используется также:

  • для команд "Фaйл -- Удалить/ Переименовать/ Копировать в папку/ Переместить в папку"
  • для команды "Фaйл -- Отправить по Email"
  • для запуска польз. команд
  • для запуска текст-конвертера

Дополнение NavPanel является бесплатным.
