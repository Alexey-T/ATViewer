Файлы настроек

Программа имеет несколько файлов настроек:

  • Portable.ini -- задает режим портабельности программы;
  • Viewer.ini -- основной файл настроек;
  • ViewerHistory.ini -- различные истории: позиция окна, строки поиска и т.д.;
  • lsplugin.ini -- настройки некоторых Lister-плагинов.

Файлы настроек ищутся в следующих папках:

  1. Папка, заданная через ключ комстроки /Ini;
  2. Папка UV, если в ней существует файл Portable.ini;
  3. Папка, записанная в реестре в "HKEY_CURRENT_USER\Software\UniversalViewer";
  4. Папка, записанная в реестре в "HKEY_LOCAL_MACHINE\Software\UniversalViewer";
  5. Низкий приоритет: папка по умолчанию "%AppData%\ATViewer".

Таким образом, чтобы сделать переносной вариант UV, необходимо создать файл Portable.ini в папке UV.
