Скрытые опции

Программа имеет несколько "скрытых" опций, не изменяемых через диалоги настроек. Чтобы изменить их, Вы должны вручную отредактировать файл настроек Viewer.ini. Для этого используйте команду "Опции -- Расширенные -- Прямое редактирование файла настроек". После сохранения файла - перезапустите программу.

Секция [Options]
FileNextMsg Команды "Файл -- Следующий / Предыдущий файл": Если установлено в 0, запрещает показ сообщений о достижении первого/последнего файла в каталоге.
FileSkipHidden Команды "Файл -- Следующий / Предыдущий файл": Если установлено в 0, скрытые файлы не будут пропускаться.
FileMoveDelay Команды "Файл -- Переименовать / Переместить в папку": Пауза (в мсек, по умолчанию 800) между закрытием файла и операцией перемещения файла. Должна быть увеличена, если не работает перемещение PDF-файлов.
StatusUrlWidth Ширина панели строки статуса, где показывается кодировка/URL (по умолчанию около 300 пикс).
ModeUndetected Номер режима, используемого для файлов неизвестного типа. По умолчанию это 1 (Текст), но Вы можете изменить это значение на 2 (Двоичный), 3 (16-ричный) или 8 (RTF/UTF-8).
ModesDisabled Список номеров режимов (числа от 1 до 8, разделенные пробелами), которые будут запрещены в процедуре авто-определения режима просмотра. Например, если этот список - "5 7 8", то при определении режима просмотра будут запрещены режимы Интернет, Плагины и RTF/UTF-8.
SearchNoCfm Если установлено в 1, можно продолжать поиск (F3) сразу же при запуске UV, используя опции поиска, сохраненные в предыдущем сеансе. Если не установлено, Вы должны будете подтвердить эти опции в диалоге поиска.
Секция [Text]
MaxClip Задает максимальный размер данных, в Мб, копируемых в Буфер Обмена в режимах Текст/Двоичный/Шестнадцатеричный/Юникод. Значение по умолчанию, 16 (Мб), позволяет избежать "задумчивости" программы при случайном выделении перед копированием слишком большого по размеру блока.
Секция [Media]
AutoAdv По умолчанию, если UV запущен с параметром @СписокФайлов, и в списке есть несколько мультимедиа-файлов, они будут проиграны по очереди. Если данная опция установлена в 0, автоматической перемотки не будет, и для перехода к следующему файлу Вам придется использовать команду "Файл -- Следующий файл".
Секция [Shortcuts]
Имя пункта меню Изменяет горячую клавишу пункта меню. Клавиши настраиваются через диалог настроек; однако, если Вам не удается назначить какую-либо клавишу через диалог, Вы можете прописать ее вручную.
  • Допустимые клавиши: Space BkSp Tab Esc Enter PgUp PgDn Home End Left Right Up Down Ins Del A...Z F1...F12.
  • Допустимые модификаторы клавиш: Shift+ Ctrl+ Alt+.
