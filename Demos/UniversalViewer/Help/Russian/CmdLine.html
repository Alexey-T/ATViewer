Параметры командной строки

Программа поддерживает следующие варианты командной строки:

  • Viewer.exe [Дополнительные параметры]
  • Viewer.exe ИмяФайла [Дополнительные параметры]
  • Viewer.exe ИмяПапки [Дополнительные параметры]
  • Viewer.exe @ИмяСписка [Дополнительные параметры]
  • Viewer.exe @@ИмяПапки [Дополнительные параметры]

Где значения параметров такие:

ИмяФайла Имя файла, открываемого при запуске.
Прим.: Имя, содержащее пробелы, надо заключить в двойные кавычки.
ИмяПапки Имя папки, открываемой при запуске в подходящем плагине просмотра папок. Пример такого плагина - VisualDirSize. Эта папка также открывается в панели навигации.
Прим.: Имя, содержащее пробелы, надо заключить в двойные кавычки.
@@ИмяПапки Указывает имя папки, содержимое которой будет прочтено при запуске и показано как список, пролистываемый командами "Файл -- Следующий файл/ Предыдущий файл". Общее число файлов будет показано в заголовке окна в квадратных скобках. Если таким образом открыто несколько медиа-клипов, они будут проиграны по очереди.
Прим.: Имя, содержащее пробелы, надо заключить в двойные кавычки.
@ИмяСписка Имя временного текстового файла (в кодировке ANSI или UTF-8 with BOM), содержащего список имен других файлов для открытия при запуске.
После успешного чтения этот файл-список будет удален.
Переходить от файла к файлу надо командами "Файл -- Следующий файл/ Предыдущий файл".
Дополнительные параметры
/Pos=Позиция Задает начальную позицию в файле.
Позиция может быть:
  • числом: абсолютное смещение. Можно указывать в режимах текстовых и Интернет.
  • числом с символом 'p': относительная позиция в процентах. Можно указывать в режимах текстовых, Интернет и Плагины.
  • числом с символом 'n': номер строки. Можно указывать только в текстовых режимах.
/P=Лев,Верх,Шир,Выс Задает начальные положение и размер окна.
Лев, Верх, Шир, Выс должны быть числами.
Значения в конце списка могут опускаться, например /P=Лев,Верх задает только положение окна.
/Print Задает печать документа после открытия. После печати - выход.
 
/Opt Вызывает диалог конфигурации программы. После этого - выход.
Прим.: параметр ИмяФайла при этом указывать не надо.
/Ini=папка Задает папку хранения конфиг. файлов.
Прим.: имя с пробелами надо заключить в дв. кавычки.
/FText=Строка
/FCase=N
/FWords=N
/FHex=N 		Задает начальную строку и опции поиска: "с учетом регистра", "целые слова", "шест. строка".
N - логическое значение (0/1).
/Mode=N Задает начальный режим просмотра.
N - номер режима (1..8).
