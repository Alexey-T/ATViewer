Введение

Universal Viewer Free - это универсальный файловый просмотрщик с большим набором поддерживаемых форматов. Доступны следующие режимы просмотра файлов:

  • Текст, Двоичный, Шестн., Юникод: любые файлы, неограниченного размера (можно просматривать файлы в том числе большие 4 Гб)
  • RTF, UTF-8: RTF и UTF-8 тексты
  • Графика: все основные форматы изображений: JPG JP2 BMP ICO GIF PNG WMF TGA TIFF...
  • Мультимедиа: форматы, поддерживаемые MS Windows Media Player: AVI MPG WMV MP3...
  • PDF, е-книги: формат PDF (не требует Adobe/Foxit Reader) + форматы XPS DjVu CBR CBZ
  • Интернет: форматы, поддерживаемые MS Internet Explorer: HTML XML MHT...
  • Плагины: форматы, поддерживаемые Lister-плагинами к Total Commander
  • MS Office: форматы пакета MS Office (если установлен): DOC DOCX XLS PPT...
  • Конвертеры: некоторые форматы в виде текста: DOC DOCX PDF ODT...

Программа полностью поддерживает Юникодные имена файлов и может интегрироваться в контекстное меню Проводника Windows: вызовите для любого файла меню правой кнопки мыши и выберите пункт "Universal Viewer". Просмотрщик также можно интегрировать в другие популярные файл-менеджеры.

В целом, программа похожа на Total Commander Lister, основные моменты по сравнению с Lister:

  • Интегрированные функции нескольких плагинов: просмотр графики, мультимедиа, веб-страниц
  • Поддержка множества кодировок: ANSI, OEM, KOI8, EBCDIC, ISO и др.
  • Поддержка текст-конвертеров
  • Поддержка пользовательских команд
  • Панель инструментов, строка статуса, другие улучшения в интерфейсе
  • Авто-обновление файла при изменении, слежение за концом файла
  • Показ номеров строк
  • Показ непечатаемых символов
  • Режим просмотра Шест./Юникод (активируйте режим Юникод дважды)
  • Современная библиотека поиска регулярных выражений
  • Предпросмотр печати
  • Просмотр EXIF-информации
