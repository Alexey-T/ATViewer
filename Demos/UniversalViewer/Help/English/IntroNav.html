NavPanel add-on

Navigation Panel add-on is called from within Universal Viewer by using F4 key (default shortcut). It allows to browse for folders and files in Explorer-like interface. The file you selected in NavPanel, is displayed in the viewer. The file you opened in the viewer, is highlighted in NavPanel too.

NavPanel is also used for:

  • "File -- Delete/ Rename/ Copy to folder/ Move to folder" commands
  • "File -- Send by Email" command
  • running user tools
  • running text converters

NavPanel add-on is freeware.
