Custom icons

To use custom icons

Copy file (in .bmp format) into "Icons" subfolder of UV installation folder. When you'll open Configure dialog, they will be available in the "Icons" dropdown list.

To create your own icons

  • Open Configure, choose "Icons" dropdown list, "Save template" item. This will create .bmp file in a directory of your choice (and also text file with hint about icons commands).
  • Open this file in a graphic editor. Resize it, so the single icon becomes of needed size. For example, default icon size is 16x16, your own size is 24x24 - so resize the bitmap to 150% to get 24 height.
  • Edit needed icons, leave others untouched. For example, if you want to change only icons visible on toolbar, then edit only these icons.
    To see what command corresponds to an icon, open the "Configure" dialog, "Shortcuts" page.
    Note: The transparent color is the color of the left-bottom bitmap px.
  • Name the resulting file "Name WWxHH (details).bmp", where "Name" is icons name, "WW"/"HH" specifies icon width/height.

You can send your files to our site. Please send only "legal" icons.
