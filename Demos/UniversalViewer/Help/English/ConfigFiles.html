Configuration files

Application has several configuration files:

  • Portable.ini -- flag-file which sets portable mode;
  • Viewer.ini -- main configuration file;
  • ViewerHistory.ini -- various histories: window position, search strings etc;
  • lsplugin.ini -- configuration of some Lister plugins.

Config files are searched in the following locations:

  1. Location specified in the command line "/Ini=" parameter;
  2. UV folder, if Portable.ini file exists there;
  3. Location stored in the registry under "HKEY_CURRENT_USER\Software\UniversalViewer";
  4. Location stored in the registry under "HKEY_LOCAL_MACHINE\Software\UniversalViewer";
  5. Lowest precedence: "%AppData%\ATViewer" folder.

To be sure that application is made portable, you need to create Portable.ini file in the app folder.
