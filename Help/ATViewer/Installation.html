Installation

First, install and enable additional libraries needed, see Additional libraries topic. After that you may install ATViewer package into Delphi/C++Builder IDE.

Package installation:

  • Delphi 5: compile and install package "Delphi\ATViewerD5.dpk"
  • Delphi 6, 7: compile and install package "Delphi\ATViewerD7.dpk"
  • Delphi 2006, 2007: compile and install package "Delphi\ATViewerD2006.dpk"
  • Delphi 2009: compile and install package "Delphi\ATViewerD2009.dproj"
  • C++Builder 5: compile and install package "CBuilder\ATViewerC5.bpk"
  • C++Builder 6: compile and install package "CBuilder\ATViewerC6.bpk"
  • C++Builder 2006, 2007: compile and install package "CBuilder\ATViewerC2006.bdsproj"

Components ATViewer, ATBinHex, ATImageBox, ATStreamSearch and ATFileNotification will be installed into the "Samples" page of component pallette. Helper components WMP and WMP9 will be installed into the "ActiveX" page of component pallette.

Note: Delphi 2005 may sometimes show "External Exception" error when application with ATViewer component closes. To prevent this, run the application outside of IDE.
