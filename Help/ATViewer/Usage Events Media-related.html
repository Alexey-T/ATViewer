Events, Media-related

OnImageLClick
OnImageRClick 		Events fired on left/right click on image control (when IsImage = True).
OnLoadImageStream Event fired after OpenStream method was called with AMode = vmodeMedia. It is of type: 
  type
    TATViewerLoadImageStream = procedure(Sender: TObject;
      AImageBox: TATImageBox; AStream: TStream) of object;
You should load image into AImageBox object from AStream parameter here.
The image can be of any type: TBitmap, TJpegImage, TMetafile etc. See example for this handler in ViewerStream demo.
