Components included

There are 5 components included in ATViewer package (on "Samples" page of component pallette), plus 2 automatically generated ActiveX controls (on "ActiveX" page of component pallette):
  1. ATViewer: main component that implements all view modes.
  2. ATBinHex: component that implements Text/Binary/Hex/Unicode view modes.
  3. ATImageBox: component that implements image scrolling, scaling and fitting.
  4. ATStreamSearch: component that performs text search.
  5. ATFileNotification: component that creates thread for watching file changes.
  6. WMP: wrapper for Windows Media Player 6.4 ActiveX.
  7. WMP9: wrapper for Windows Media Player 9.0 ActiveX.
