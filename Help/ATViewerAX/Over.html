ATViewer OCX

ATViewer OCX is ActiveX version of ATViewer component. This control allows to view multiple file types. Most of features of original component are available.

The following file types are supported:

  • PDF: PDF files supported w/o need of installed Adobe/Foxit Reader (but helper dll is needed; included).
  • Images: lot of image formats: jpeg, gif, png, bmp, wmf, emf, tiff (multi-paged), pcx, tga, etc.
    Image can be scrolled, zoomed, panned by mouse. Image can be fit to window, centered in window. Print/preview is avaialble. Resampling for scaled size is available.
  • Multimedia: lot of media formats: mp3, mpg, avi, wav, midi, mp4, wma, wmv, etc.
    View can be based on Windows Media Player engine, or directly on DirectX engine. Installed codecs for formats are required.
  • Web: web pages: html, php, xml, mht, etc.
    View is based on IE engine. IE methods available.
  • Text, Binary, Unicode: all files.
    Only viewable portion of file is loaded into memory, 4Gb+ sizes are allowed. Lot of encodings supported: all 1-byte, Unicode LE/BE, MBCS (Asian), UTF8 etc. Additional features available: gutter/line numbers, auto-reloading/tailing, searching, Hex viewer, print/preview etc.
  • RTF/UTF-8: RTF, UTF-8 texts.
    Print, copying to clipboard, search methods are avaialble.
  • Plugins.
    Support for Total Commander Lister plugins is avaialble. Plugins' methods (print, copy to clipboard...) can be called.
