Events, Media-related

OnMediaPlaybackEnd Event fired after media playback has been finished (when IsMedia = True).
OnImageLClick
OnImageRClick 		Events fired after left/right mouse clicks happen, over the image (when IsImage = True).
OnImageMouseDown
OnImageMouseUp
OnImageMouseMove 		Events fired when mouse button pressed/released, or mouse moved, over the image (when IsImage = True).
Parameters:
  • Btn gives mouse button number. 0: left, 1: right, 2: middle. (Not used for OnImageMouseMove.)
  • Shift gives control keys state. Bit 0: Shift pressed, bit 1: Alt pressed, bit 2: Ctrl pressed, bit 3: left mouse button pressed, bit 4: right mouse button pressed, bit 5: middle button pressed.
  • X and Y give image clicked position. You may need to get ImageWidth/ImageHeight/ImageScale, then you may calculate which image pixel is clicked.
