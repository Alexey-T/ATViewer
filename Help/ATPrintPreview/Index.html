Introduction

ATPrintPreview component allows to quickly implement "Print preview" dialog for image viewing applications. You can call preview form for a TBitmap/TImage with a single function call.

On this form user can change some options and start the actual printing. Note that the preview page on the right is adapted to match the active printer's page sizes.

Functions implement the following options:

  • Fit mode (normal, best fit, stretch...)
  • Margins
  • Footer text
  • Image position
  • Image gamma correction
